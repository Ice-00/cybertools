# Local Password Strength & Entropy Validator

A lightweight, local Python tool designed to evaluate password strength against corporate policy rules without sending sensitive credentials over the internet.

## 🔒 Security Features
* **Zero-Network Footprint:** Runs entirely locally to ensure passwords are never intercepted or logged externally.
* **Length Verification:** Ensures adherence to minimum 8-character baseline requirements.
* **Character Diversity Analysis:** Validates numerical density and uppercase distribution to mitigate brute-force vulnerabilities.

## 🚀 How to Run
Ensure you have Python 3 installed. Clone the repository and run the script from your terminal:

```bash
python password_check.py

🛠️ Requirements

    Python 3.x (No external dependencies required)

⚖️ Disclaimer

This tool is intended strictly for educational and local administrative purposes to analyze credential strength.
